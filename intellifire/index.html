Welcome to IntelliFire DSS

IntelliFire is the Next-Level Decision Support System designed to facilitate wildfire management locally, nationaly and Globaly

Design Features

  • The Base is Clean Architecture
  • designed using the SOLID approach
  • Separation of code in different layers with assigned responsibilities making it easier for further modification
  • High Level if Abstraction
  • Loose Coupling of code
  • Painless and Effective testing strategy that follows the testing pyramid
  • Impeccable seperation of Business logic and machinery - Frameworks are isolated in individual modules - Changes only have to change one place, with the rest of the app not even knowing about it or caring
  • The app has use cases rather than being tied to a CRUD system
  • Screaming architecture a.k.a. it screams its intended usage. When you look at the package structure you get a feel for what the application does rather than seeing technical details
  • All business logic is in a use case so it's easy to find and it's not duplicated anywhere else
  • uses a microservices architechture to separate APIs by entity.
