◆ stopJobPriority()

stopJobPriority (   $jobname,
  $priority 
)

Requests that a specific job stop executing.

Parameters
string$jobnameThe job to stop executing.
int$priorityThe priority with which to stop the job. 0 -> Stop at the soonest time available (may not occur until currently running simulations have completed). 1 -> Stop at the soonest time available but attempt to terminate the job if still running after 5 minutes. 2 -> Attempt to terminate the executing process immediately.
