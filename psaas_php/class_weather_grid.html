Parameters
String$filenameThe location of the grid file.
String$projectionThe projection file.
int$sectorThe sector (wind direction) to apply this grid file to. Must be one of the WeatherGrid_GridFile::SECTOR_* values. Only one of each sector is allowed per station.
float$speedThe wind speed.
Exceptions
ExceptionThrows an exception if a file for the same sector has already been added.
