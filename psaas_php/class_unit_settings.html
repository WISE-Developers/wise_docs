Detailed Description

Settings that define which units will be used when data is exported in summary or statistics files.

All units are optional with application defaults being used for anything that isn't specified. It is the users responsibility to make sure only unit values that are valid for the type of units being displayed are set. Anything that is invalid will cause undefined behaviour.
