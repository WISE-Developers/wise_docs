Add a file attachment to the project.

Attachments can be used anywhere a filename would be used.

Parameters
string$filenameThe name of the file to attach. Must be a valid Windows filename.

◆ streamOutputToGeoServer()

streamOutputToGeoServer (   $username,
  $password,
  $url,
  $workspace,
  $coverageStore,
  $srs 
)

Stream output files to a GeoServer instance.

Currently only GeoTIFF files can be streamed to GeoServer.

Parameters
string$usernameThe username to authenticate on the GeoServer instance with.
string$passwordThe password to authenticate on the GeoServer instance with. WARNING: the password will be stored in plain text.
string$urlThe URL of the GeoServer instance. The web interface should be at {url}/web.
string$workspaceThe name of the workspace to upload the file to. If the workspace doesn't exist it will be created.
string$coverageStoreThe name of the coverage store to upload the file to. If the coverage store doesn't exist it will be created.
string$srsThe declared spatial reference system of the uploaded file. If not provided the uploaded coverage will not be enabled.
