start

  • start(options?: IJobManagerOptions): Promise<void>

  • Connect to the MQTT broker and start listening for messages. If the client has already connected this method will do nothing.

    example

    let client = new JobManager("job_id"); client.start({ host: "mqtt.hostname", port: 1883, clientId: "unique.client.id", //will be generated if not present username: "username", password: "password" });

    Parameters

    • Optional options: IJobManagerOptions

      An optional collection of options to modify how the MQTT client connects to the broker. Options only need to be specified if they are different from the default values.

    Returns Promise<void>

  • setDefaults(options: IJobManagerOptions): void

  • Set the default broker connection options. If no options are specified when {@link JobManager#start} is called these values will be used. {@link IJobManagerOptions#clientId} will always be ignored as it should be unique between multiple connections.
