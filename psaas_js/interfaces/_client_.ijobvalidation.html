Interface IJobValidation

A message that will be sent after a job has been validated by PSaaS.

Hierarchy

Properties

error_list

error_list: IJobValidationError[]

A list of errors and warnings that were found during validation.

load_warnings

load_warnings: string

A user readable string that indicates some of the default updates that have been made in PSaaS that affect the simulation.

success

success: boolean

Was the validation successfully run. This will only be false if PSaaS couldn't be started or the validation couldn't even be run, it doesn't indicate that the job is or isn't valid.

valid

valid: boolean

Was the FGM valid. This will be true if the FGM is valid and ready to be run, even if there were warnings. If there were errors generated by the validation this will be false.
