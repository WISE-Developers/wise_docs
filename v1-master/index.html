PSaaS

PSaaS is an Acronym for "Prometheus Software as a Service". Prometheus is the Canadian Wildland Fire Growth Simulation Model and PSaaS is a modelling framework based on Prometheus. There are four APIs used to write PSaaS models:

  • Javascript
  • .NET
  • MQTT
  • PHP
  • Java

IntelliFire

IntelliFire is a combination server micro-services that provide Wildfire Decision Support Services to a web based client, EasyMap. IntelliFire replaces the Drupal™ Based S.P.A.R.C.S Platform that has been the primary Decision Support System at the Government of the Northwest Territories since 2003. Microservices are written in Node JS Javascript. Documentation is in the JSDOC format

EasyMap

EasyMap is a web application with a NodeJS foundation and an HTML/JavaScript GUI. It is a web based client that consumes and uses IntelliFire and SPARCS Microservices. Documentation is in the JSDOC format
